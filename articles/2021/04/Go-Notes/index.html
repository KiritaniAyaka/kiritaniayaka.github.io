Go Notes

|字数总计:284|阅读时长:1分钟|阅读量:

基础

导包

使用 import 包路径 导入包；使用 import _ 包路径 则只会执行包中的init()函数而不会导包。

变量

声明格式： var name type

声明变量并初始化： var name [type] = value （初始化的时候类型可以省略，会自动匹配变量类型）

在函数内部，可以使用更简略的方式声明并初始化变量： name := value

匿名变量

使用多重赋值或接受多个返回值的函数时使用 _ 来忽略不需要的值。

常量

常量使用 const name = value 来声明。

iota

iota是Go语言中的常量计数器，在const出现的时候iota被初始化为0，每一行iota的值增加1。在枚举的时候可以使用 _ 跳过某些值。

数组

数组使用 var name [len]type 来定义，数组长度也是类型的一部分。数组在声明的时候必须指明长度，且长度一旦指定就不可改变。

数组是值类型，在Go语言中使用数组传参会复制整个数组。

切片

使用 var name []type 来创建一个切片
