Eclipse调整控制台和编辑器字体

发表于2020-11-12|更新于2020-11-12|编程
|字数总计:192|阅读时长:1分钟|阅读量:

最近学校专业课学到了eclipse，第一个程序是打印一个DOS菜单，然后发现两个半角和一个全角的宽度不一样，于是需要修改编辑器和控制台的字体才能解决。修改方法如下。

  1. 在编辑器中右键，并进入 Preferences… 菜单项。

  1. 在搜索栏中输入 font ，在搜索结果中点击 Colors and Fonts 项目。

  1. 在右边展开 Basic ，选择下面的 Text Font 并点击右边的 Edit 。

    如果调整的是编辑器内字体，则展开 Java ，选择 Java Editor Text Font 后续操作相同。

  1. 在弹出的对话框中更改自己想要的字体样式以及字体大小即可。

    这里推荐使用 Courier New 等等宽字体。
